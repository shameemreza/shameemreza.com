## How to install GeneratePress Premium?


https://vimeo.com/189862256

[GeneratePress](https://generatepress.com) is a free theme, but the paid version is a **plugin** called **GP Premium** which, when installed, unlocks all the features that make this theme so unique.

Being a plugin, some users get confused and try to install it as if it were a WordPress theme, and of course, they get an error. I have already come across several such cases.

Since GP Premium is a plugin, You need to install GeneratePress Premium as you install a plugin in WordPress, not as a [WordPress theme](/10-keys-to-choose-the-best-wordpress-theme/).

Once the GP Premium plugin is installed, you have to go to **Appearance** -> **GeneratePress** to activate the license and the addons you will use.
